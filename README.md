# How Computers Learn to Make Decisions: The Power of Conditional Statements

This project explores how computers make decisions using conditional statements in programming. These are the building blocks of logic that allow programs to react differently based on different inputs and conditions.

 Types of Conditional Statements
1. if Statement

Executes a block of code only when the condition is true.

2. if-else Statement

Handles both outcomes:

Runs one block if the condition is true
Runs another block if the condition is false
3. elif Ladder

Used when there are multiple conditions to check step by step.

4. Nested if

An if statement placed inside another if, used for complex decision-making logic.

5. Ternary Operator

A short-hand version of if-else, used for simple conditional expressions.

 Purpose of This Project
Understand how decision-making works in programming
Learn different types of conditional statements
Build a strong foundation for problem-solving in Python
