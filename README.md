# How Computers Learn to Make Decisions: The Power of Conditional Statements

Understanding types of Types of Conditional Statements

• IF Statement → Executes when condition is true 
• IF-ELSE Statement → Handles both true and false cases 
• ELIF Ladder → Checks multiple conditions step by step 
• Nested IF → IF inside another IF for complex logic 
• Ternary Operator → Short form of IF-ELSE
